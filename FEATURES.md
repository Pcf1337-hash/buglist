# BugList – Features

## Schulden tracken
- Personen anlegen (Name, Telefon, Notizen)
- Schulden hinzufügen: wer schuldet wem wie viel
- Richtung wählbar: „Schuldet mir" oder „Ich schulde"
- Beträge über ein einhand-bedienbares Nummernpad eingeben
- Schnell-Buttons (+10 / +50 / +100 usw.) für schnelle Betragseingabe

## Teilzahlungen
- Teilzahlungen auf offene Schulden buchen
- Fortschrittsbalken zeigt wie viel bereits bezahlt wurde
- Status wird automatisch gesetzt: OFFEN → TEILWEISE → BEZAHLT

## Tags
- Schulden mit Tags versehen statt Freitext (z. B. Hase, Cal, €, K)
- Tags in den Einstellungen selbst verwalten (hinzufügen & löschen)

## Dashboard
- Übersicht aller Personen mit Gesamtbilanz
- Farbcodiert: Grün = schuldet mir, Rot = ich schulde
- Sortierung nach Name oder Betrag

## Statistiken
- Monatliche Auswertung der Schulden
- Diagramme über Ein- und Ausgaben

## Einstellungen
- Währung wählbar
- Auto-Lock nach Inaktivität
- Alle Daten exportieren (CSV)
- Alle Daten löschen

## Sicherheit
- Biometrische Authentifizierung beim Start (Fingerabdruck / Gesicht)
- Datenbank vollständig verschlüsselt (SQLCipher AES-256)
- Kein Cloud-Backup, alle Daten bleiben auf dem Gerät
- Kein Screenshot möglich (FLAG_SECURE)

## Design
- Dunkles Design (Dark Mode)
- Gold als Akzentfarbe
- Läuft auf Android 8.0 und neuer
